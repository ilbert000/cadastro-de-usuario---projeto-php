# CRUD de Usuários em PHP Puro

Sistema CRUD simples com MVC e Bootstrap.

## Estrutura
- config/database.php
- controllers/UserController.php
- models/User.php
- views/users/index.php
- views/users/create.php
- views/users/edit.php
- routes/web.php
- public/index.php

## Como usar
1. Crie o banco e tabela com `database/users.sql`.
2. Ajuste `.env` se necessário.
3. Abra `public/index.php` no navegador (pelo servidor local).

## Funcionalidades
- Listar usuários
- Criar usuário
- Editar usuário
- Excluir usuário
